Müşteri Ekle

Her oyunda kullanılabilen drop-in Unity SDK. Aşağıdaki adımları sırayla takip et — 5 dakikada veri akmaya başlar. SDK otomatik olarak oturum, performans ve crash verilerini toplar.

1. SDK'yı Projene Ekle

Oyun ekledikten sonra "Oyunlarım" sekmesinde her oyunun yanında "SDK İndir" butonu çıkar. İndirdiğin zip içinde:

  • AltareAnalytics.cs — Unity drop-in SDK (Assets klasörüne at)
  • AltareAnalyticsBootstrap.cs — otomatik başlangıç + KVKK/GDPR consent (gameId pre-filled)
  • AltareConfig.json — gameId + ayarlar pre-filled
  • SampleUsage.cs — örnek event çağrıları
  • KURULUM_REHBERI_TR.txt / SETUP_GUIDE_EN.txt — adım adım kurulum (TR + EN)

Firebase Unity SDK gerekli: Authentication + Firestore modüllerini resmi siteden indir, Unity'e import et.

2. Bootstrap — Tek Satırlık Initialize

Yol A (Önerilen): AltareAnalyticsBootstrap.cs dosyasını projeye at — başka bir şey yapma. Bootstrap, KVKK/GDPR onayını kontrol eder ve SDK'yı otomatik başlatır. GameId ve GameName zip'te pre-filled gelir.

Yol B (Manuel): Bootstrap kullanmıyorsan sahnenin en üst MonoBehaviour'ında:

using Altare.Analytics;

void Start() {
    AltareAnalytics.Initialize("YOUR_GAME_ID", "Your Game Name");
}

SDK otomatik olarak sessionId üretir. Her oturum ve app resume'da yeni bir session başlatılır — panelde Aktif Oturum sayısı buna göre ölçülür. Ayrıca first_open ve app_open event'lerini otomatik gönderir.

3. Event Çağrıları — Standart Taxonomy

// ── Level olayları ──
AltareAnalytics.LogEvent("level_start",    new() { { "level", level } });
AltareAnalytics.LogEvent("level_complete", new() { { "level", level }, { "score", score } });
AltareAnalytics.LogEvent("level_fail",     new() { { "level", level }, { "moves_used", moves } });

// ── Reklam ──
AltareAnalytics.LogEvent("ad_watched",          new() { { "placement", "interstitial" } });
AltareAnalytics.LogEvent("rewarded_ad_watched",  new() { { "placement", "extra_lives" } });

// ── IAP (Uygulama İçi Satın Alma) ──
AltareAnalytics.LogEvent("iap_purchase_success", new() { { "sku", sku }, { "amount_usd", price } });

// ── Player feedback ──
AltareAnalytics.SubmitFeedback(rating: 5, text: "Harika oyun!");
Otomatik Event'ler (manuel çağırma yok):
  • session_start — uygulama açılışında ve resume'da
  • session_end — uygulama pause/quit'te (duration_seconds dahil)
  • fps_warning — FPS <30 düşünce (5sn aralıkla kontrol, 60sn cooldown)
  • crash_detected — beklenmeyen kapanmalarda

4. Firebase Setup (her oyun için bir kere)

  1. Firebase Console → projeye Android app ekle (paket adı eşleşmeli)
  2. google-services.json dosyasını Assets/ köküne koy
  3. Authentication → Sign-in method → Anonymous provider'ı Enable yap
  4. Firestore Database → Rules'da oyunun event yazmasına izin ver (varsayılan kurallar bunu destekler)
  5. Build → telefona kur → ilk açılışta event akmaya başlar

5. Doğrulama Checklist

  1. Oyunu telefonda aç, 1-2 dakika oyna
  2. "Canlı Event Stream" sekmesinde session_start event'i görünmeli
  3. Level başlat/bitir → level_start + level_complete akmalı
  4. "Genel Bakış"'ta Aktif Oturum > 0 olmalı
  5. Reklam izlet → ad_watched event'i geldiğini onayla
  6. 24 saat veri biriktikten sonra "AI Raporu" üretebilirsin

Sorun mu var? Unity Console'da [Altare] ile başlayan logları kontrol et. [Altare] Ready. uid=... mesajı görmüyorsan Firebase SDK kurulumunu tekrar kontrol et. google-services.json eksik veya paket adı uyumsuz olabilir.

6. Her Event'te Gönderilen Veriler

SDK her event'te şu alanları otomatik doldurur — ek bir şey yapmanıza gerek yok:

gameId Oyun kimliği (initialize'da verilen)
sessionId Her oturum için benzersiz UUID (otomatik üretilir)
playerAnonId Anonim oyuncu kimliği (cihazda saklanır, PII yok)
platform Android / iOS / Editor
appVersion Oyun sürümü (Application.version)
deviceModel Cihaz modeli (Samsung SM-A217F vb.)
Gerçek Play Store rakip + yorum verisinden AI yeni oyun konseptleri üretir. Önce Pazar Analizi sekmesinden ilgili kategori için rakipleri çek, sonra burada "Konsept Üret" bas.
