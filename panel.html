Müşteri Ekle

Royal Dreams + Crimson Scar dahil her oyunda kullanılabilen drop-in Unity SDK. Aşağıdaki adımları sırayla takip et — 5 dakikada veri akmaya başlar.

1. SDK'yı Projene Ekle

Oyun ekledikten sonra "Oyunlarım" sekmesinde her oyunun yanında "SDK İndir" butonu çıkar. İndirdiğin zip içinde:

  • AltareAnalytics.cs — Unity drop-in script
  • AltareConfig.json — gameId pre-filled
  • INTEGRATION_GUIDE.pdf — adım adım rehber
  • SAMPLE_USAGE.cs — örnek event çağrıları

Firebase Unity SDK gerekli: Authentication + Firestore modüllerini resmi siteden indir, Unity'e import et.

2. Bootstrap — Tek Satırlık Initialize

using Altare.Analytics;

// Bootstrap script (sahnenin en üst MonoBehaviour'ında)
void Start() {
    AltareAnalytics.Initialize("YOUR_GAME_ID", "Your Game Name");
}

YOUR_GAME_ID — Oyunlarım sekmesinde sistemin sana verdiği değer (örn: altare.royal-dreams).

3. Event Çağrıları — Standart Taxonomy

// Level olayları
AltareAnalytics.LogEvent("level_start",    new() { { "level", level } });
AltareAnalytics.LogEvent("level_complete", new() { { "level", level }, { "score", score } });
AltareAnalytics.LogEvent("level_fail",     new() { { "level", level }, { "moves_used", moves } });

// Reklam + IAP
AltareAnalytics.LogEvent("ad_watched",     new() { { "placement", "continue" } });
AltareAnalytics.LogEvent("rewarded_ad_watched", new() { { "placement", "extra_lives" } });
AltareAnalytics.LogEvent("iap_purchase_success", new() { { "sku", sku }, { "amount_usd", price } });

// Player feedback
AltareAnalytics.SubmitFeedback(rating: 5, text: "Harika oyun!");

SDK session_start, session_end, fps_warning, crash_detected event'lerini otomatik yazar — manuel çağırmana gerek yok.

4. Firebase Setup (her oyun için bir kere)

  • Firebase Console → projeye Android app ekle (paket adı eşleşmeli)
  • google-services.json dosyasını Assets/ köküne koy
  • Authentication → Anonymous Provider'ı Enable yap
  • Build → telefona kur → ilk açılışta event akmaya başlar

5. Doğrulama

Oyunda 1-2 dakika oyna, "Canlı Event Stream" sekmesinde session_start + level_start event'leri akmaya başlamalı. "AI Raporu" sekmesinde 24 saat veri biriktikten sonra ilk raporunu üretebilirsin.

Öneri 1 · Cozy Match-3 + Decoration Hibrit High Potential

Kategori trend sinyalleri "cozy" (+%34) ve "decoration" (+%28) temalarının yorumlarda yükseldiğini gösteriyor. Royal Match'in formülünü takip et ama farklı tema (köy/cottage). Royal Dreams'in match-3 kodunu reuse edebilir.

Tahmini development: 4-5 ay · Market window: Q3 2026

Öneri 2 · Crimson Scar Live-Ops Medium

Crimson Scar mid-core dungeon yapısı; mid-core kategoride "PvP arena" yorumlarda parlıyor. Mevcut dungeon engine'in üstüne haftalık event arenası eklenebilir. Düşük efor, retention etkisi yüksek.

Öneri 3 · Block Puzzle Re-Theme Low Effort

Block Blast! yorumlarında "tema sıkıcı" şikayeti (%14). Mevcut block puzzle engine'imiz üzerine 3 farklı tema (woody, neon, ocean) hızlı re-theme yapılabilir.
